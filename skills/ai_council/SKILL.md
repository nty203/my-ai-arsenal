# AI 평의회 (AI Council) 스킬 가이드

당신은 이 스킬을 설치한 에이전트입니다. 이 스킬은 복잡한 문제나 다각적인 시각이 필요할 때 외부 LLM들의 의견을 가져와 당신의 판단에 융합할 수 있도록 돕습니다.

## 🛠️ 트리거 및 사용 시점
사용자가 다음과 같이 지시할 때 자동(또는 수동)으로 구동하세요.
- "이 문제에 대해 ChatGPT와 Claude의 의견도 들어보고 종합해줘"
- "다중 AI 토론 모드 발동해"

## 🚀 실행 절차

1. **프롬프트 저장**: 사용자의 질문/안건을 텍스트 파일(`council_prompt.txt`)로 현재 작업 폴더에 저장합니다.
2. **백그라운드 스크립트 실행**: 파이썬 환경에서 다음 명령어를 실행하여 외부 의원(ChatGPT, Claude)을 소집합니다.
   ```bash
   python council_runner.py -p council_prompt.txt -o council_result.md
   ```
   *(이 스크립트는 Selenium 기반 가상 타이핑 기술을 사용해 방어벽을 우회합니다. 봇 차단 방지를 위해 사용자가 최초 1회 `setup_login.py`를 실행하여 쿠키를 저장했음을 전제로 합니다.)*
3. **결과 분석 및 종합**: 스크립트 실행이 성공적으로 끝나면 `council_result.md`에 담긴 타 AI들의 답변을 읽어들입니다.
4. **최종 판단 제출**: 타 AI의 의견 중 유용한 부분을 발췌하여 당신만의 논리와 결합한 '삼위일체 최종 보고서'를 사용자에게 제출합니다.
